Welcome to Sonam's Authorization Server manager

This is the Authorization Manager web application.

This is for managing OAuth2 Client configuration and for setting roles with organization. You can sign-up and create an account to create OAuth2 client and manage roles.

This AuthzManager and the Authorization Server supports a single tenant authorization server.
